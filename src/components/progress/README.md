# Progress 组件

*本组件的目的就是将elementUi组件的progress组件添加动画效果，仅此而已*

## props

| prop name    | description  | type      | values | default / require |
| ------------ | ------------ | --------- | ------ | ----------------- |
| percentage   | 进度百分比   | `Number`  | -      | `required`        |
| isAnimattion | 是否显示动画 | `Boolean` | -      | `false`           |
| time         | 动画时长     | `Number`  | -      | `2000`            |
